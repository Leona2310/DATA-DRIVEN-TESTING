DATA-DRIVEN TESTING USING SELENIUM (CODTECH TASK)

SETUP INSTRUCTIONS:

1. Install Python packages:
   pip install selenium pandas

2. Download ChromeDriver matching your Chrome version:
   https://chromedriver.chromium.org/downloads

3. Place chromedriver.exe inside this folder.

4. Run the script:
   python data_driven_test.py

OUTPUT:
results.txt will contain PASS / FAIL results.
